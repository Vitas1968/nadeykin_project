# Claude Code instructions

Работай как помощник по разработке проекта анализа тендерных документов.

## Основные правила
- Объясняй кратко и по делу на русском языке.
- Перед изменением файлов показывай план.
- Не трогай секреты, .env и credentials.
- Не удаляй документы пользователя.
- Не изменяй исходные tender-документы без явной команды.

## Технические правила
- Предпочитай Python.
- Код должен быть простым и проверяемым.
- Для чтения docx/xlsx/pdf/html использовать отдельные reader-модули.
- Логику анализа не смешивать с логикой чтения файлов.
